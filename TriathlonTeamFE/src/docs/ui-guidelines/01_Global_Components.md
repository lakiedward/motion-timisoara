## Quick Reference

### Butoane
| Clasă | Utilizare |
| :--- | :--- |
| `.btn-elite--primary` | Gradient albastru, pill shape |
| `.btn-elite--outline` | Border albastru |
| `.hero-btn` | Pentru page heroes |
| `.icon-btn` | 36x36px, pentru acțiuni |
| `.toggle-btn` | Activate/Deactivate |

### Badges
| Clasă | Zonă/Status |
| :--- | :--- |
| `.badge--warning` | Amber (Admin) |
| `.badge--success` | Verde (Activ) |
| `.badge--error` | Roșu (Eroare) |
| `.badge--coach` | Verde (Coach) |
| `.badge--club` | Indigo (Club) |
| `.badge--primary` | Albastru (Public) |

### Carduri
| Clasă | Utilizare |
| :--- | :--- |
| `.stat-card` | Statistici |
| `.entity-card` | Liste entități |
| `.section-card` | Container secțiuni |

### Grids
- `.entity-grid` - `minmax(320px, 1fr)`
- `.stats-row` - `minmax(200px, 1fr)`
